# АТАКА — Реалізований функціонал

## Авторизація

**SMS OTP:**
- Вхід по номеру телефону (+380)
- 6-значний код, дійсний 5 хвилин
- Ліміт 5 запитів на годину
- Захист від брутфорсу (3 спроби)

**Google OAuth:**
- Вхід через Google акаунт
- Автоматичне створення профілю

**Токени:**
- Access token — 7 днів
- Refresh token — 30 днів

## Ролі

| Роль | Доступ |
|------|--------|
| PARENT | Діти, оплати, розклад, змагання |

## Мультитенантність (SaaS)

**Tenant (Клуб):**
- Унікальний slug
- Назва, бренд, логотип
- Кольори брендингу
- План: START / PRO / AI
- Ліміти: учні, тренери, зали
- Статус: активний/неактивний

**Плани:**
| План | Ціна | Учні | Тренери |
|------|------|------|---------|
| START | 990 ₴ | 50 | 3 |
| PRO | 2490 ₴ | 200 | 10 |
| AI | 4990 ₴ | ∞ | ∞ |

**Super Admin панель:**
- Список всіх клубів
- MRR (загальний дохід)
- Створення клубів
- Зміна планів
- Активація/деактивація

---

## Структура клубу

**Location (Зал):**
- Назва, адреса, місто, район
- Максимальна місткість
- Статус активності

**Program (Програма):**
- Тип: KIDS / SPECIAL / SELF_DEFENSE / MENTORSHIP / CONSULTATION
- Ціна за місяць
- Тренувань на тиждень
- Тривалість заняття
- Вікові обмеження
- Рівень складності

**Group (Група):**
- Прив'язка до програми та залу
- Призначений тренер
- Розклад занять

**Child (Учень):**
- ПІБ, дата народження
- Група, тренер, батько
- Поточний пояс
- Прогрес до наступного пояса
- Медалі та бали
- Ціль відвідувань (12/міс)

---

## Система поясів

WHITE → YELLOW → ORANGE → GREEN → BLUE → PURPLE → BROWN → BLACK

**Логіка:**
- Тренер ставить оцінки → накопичується прогрес
- При 100% тренер схвалює підвищення
- Адмін присвоює новий пояс

---

## Відвідуваність

**Статуси:**
- PRESENT — присутній
- ABSENT — відсутній
- LATE — запізнився
- WARNED — попередження
- EXCUSED — поважна причина

**Функції:**
- Тренер відмічає групу
- Коментарі до кожного учня
- Автоматичний % відвідуваності
- Підрахунок серії пропусків
- Тригер MetaBrain при 3+ пропусках

---

## Біллінг

**Subscription (Підписка):**
- Прив'язка до дитини та батька
- Ціна, валюта (UAH)
- Цикл: MONTHLY
- День оплати (1-28)
- Статус: ACTIVE / PAUSED / CANCELLED
- Дата наступного списання

**Invoice (Рахунок):**
- Автогенерація по підписці
- Сума, опис, дедлайн
- Статус: PENDING / PAID / OVERDUE / CANCELLED
- Дані WayForPay транзакції
- Застосована знижка

**CRON задачі:**
- Щоденна генерація invoices
- Маркування прострочених
- Нагадування про оплату

## WayForPay

**Статус:** Тестовий режим (test_merch_n1)

**Реалізовано:**
- Генерація підпису HMAC_MD5
- Redirect на сторінку оплати
- Webhook обробка callback
- Оновлення статусу invoice
- Push-сповіщення після оплати
- Симуляція оплати (для тестів)

**Тестова картка:** 4111111111111111

## Знижки

**Типи:**
| Тип | Опис |
|-----|------|
| REFERRAL | За запрошення (50%) |
| FAMILY | 2+ дітей (10-20%) |
| LOYALTY | 3+ міс (5-10%) |
| FIRST_TIME | Перша оплата (10%) |
| PROMO | Промокод |
| METABRAIN | Персональна AI |

**Параметри знижки:**
- Тип значення: PERCENT / FIXED / FREE_PERIOD
- Пріоритет
- Можливість комбінування
- Ліміт використань
- Ліміт на користувача
- Термін дії
- Мінімальна сума покупки
- Максимальна знижка

**Правила:**
- Сортування по пріоритету
- Максимум 50%
- Групова ексклюзивність

## Реферальна система

**Механіка:**
- Кожен користувач має referralCode
- Новий клієнт вводить код при реєстрації
- Після першої оплати:
  - Запросивший: -50% на наступний місяць
  - Запрошений: -10% на перший місяць

**Статуси referral:**
PENDING → REGISTERED → CONFIRMED → REWARDED

## MetaBrain (AI)

**Risk scoring (0-100):**
- 0-10: VIP
- 11-39: ACTIVE
- 40-69: WARNING
- 70-100: CHURN_RISK

**Фактори:**
| Фактор | Вага |
|--------|------|
| Відвідуваність | 30% |
| Тренд відвідувань | 15% |
| Статус оплати | 25% |
| Прогрес | 15% |
| Серія пропусків | 15% |

**Actions для тренера:**
- Зателефонувати батькам
- Написати повідомлення
- Похвалити на занятті
- Запропонувати знижку

**Автоматичні знижки:**
- Створюються для CHURN_RISK та WARNING
- TTL 48-72 години
- Одноразове використання

## LTV Engine

**LTV Profile:**
- Загальна сума оплат
- Місяців активності
- Середня оплата/міс
- Прогнозований LTV

**Сегменти:**
| Сегмент | LTV | Знижка |
|---------|-----|--------|
| GOLD | >30k | 10% |
| SILVER | 15-30k | 15% |
| BRONZE | 5-15k | 25% |
| NEW | <5k | 30% |

**Логіка:**
- isWorthSaving: LTV < 2000 AND risk > 80 = не рятувати
- isUpsellCandidate: LTV > 15000 AND risk < 30 = VIP оффер
